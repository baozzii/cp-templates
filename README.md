# cp-templates

一个基于Go的算法竞赛模板库。（鸣谢灵神）

## 一些约定

- 本模板注重泛用性，即将需要修改的部分尽量抽象成接口，从而满足大部分需求。由于Go中的泛型系统并不是非常完善，且泛型会有一定的运行时开销，本模板的性能可能不会是最快的，但保证会控制在一个合理的范围内（最多为C++较快实现的**2倍**）。

- 除特殊注明外，所有涉及到的“下标”都为 **$0$ 下标**；所有涉及到的“区间”都为**左闭右开**。

- 保证可以在Codeforces, Leetcode, Atcoder, Luogu中**最低版本**的Go中运行，具体版本号可以查看`go.mod`文件。

- 模板默认`int`和`uint`类型为`64`位。

- Go对算法竞赛天生支持度较低，所以作者造了很多轮子（如`Vec`对应C++中的`vector`，`Umap`对应C++中的`unordered_map`等等）。虽然无法一比一复现（如Codeforces的Go版本目前还不支持迭代器），但已经在允许范围内做到较为还原。

## 注意事项

- 为了追求一些性能，某些模板使用了C语言实现（如矩阵乘法）。具体使用方法可见 `template/README.md`。
